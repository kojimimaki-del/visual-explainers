3HTTP メソッド（CRUD との対応）
REST API で使われる 5 つの代表的メソッドと用途
メソッド 意味 CRUD 例 特徴
GET リソースを取得 Read GET /users/42 べき等・安全。ボディなし
POST リソースを新規作成 Create POST /users 非べき等。レスポンス 201 Created
PUT リソースを完全置換 Update PUT /users/42 べき等。未指定フィールドは null になる
PATCH リソースを部分更新 Update PATCH /users/42 差分のみ送る。PUT より軽量
DELETE リソースを削除 Delete DELETE /users/42 べき等。レスポンス 204 No Content
